# Analisis del Mercado Chileno de Seguros de Vida
## Proyecto BI basado en estructura CMF
### Proyecto de portafolio desarrollado con datos 100% sintéticos inspirados en la estructura financiera y operacional del mercado asegurador chileno. No contiene información real, privada ni confidencial de ninguna compañía.
----
## **Fase 1 — Problema de negocio**

**Escenario:** la gerencia de una aseguradora multinacional de vida necesita comprender la evolución del mercado, identificar competidores con mayor crecimiento y evaluar indicadores de rentabilidad, eficiencia, siniestralidad y solvencia. El análisis se realizará trimestralmente entre 2023 y 2025.

**Preguntas de negocio:**
- ¿qué compañías tienen mayor participación?,
- ¿cuáles están creciendo?
- ¿qué compañías presentan mayor siniestralidad?
- ¿qué tan eficientes son sus gastos?
- ¿qué negocios concentran las primas?
- ¿qué compañías presentan una posición de solvencia más sólida?
